# Stylesheet Structure

- `base.css`: global reset, tokens, base page styles, nav, buttons, shared section layout
- `hero.css`: front-page hero composition and name/photo blend styling
- `about.css`: dark editorial About section styles
- `sections.css`: skills, projects, education, certifications, contact, footer
- `interactions.css`: hover states for pointer devices
- `animations.css`: keyframes and reveal animation helpers
- `responsive.css`: breakpoints and reduced-motion adjustments

`../style.css` is the entry file that imports these modules in the correct order.
